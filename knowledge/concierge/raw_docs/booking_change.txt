Concierge Booking Change SOP

1. Confirm caller identity with registered phone and booking reference.
2. Booking date can be changed up to 24 hours before appointment time.
3. Same-day changes are not allowed unless manager exception is approved.
4. Cancellation procedure requires documenting reason code and sending confirmation email.
5. For service issue troubleshooting, create a support case and set priority based on customer impact.
